### 1. == 연산자  
1. 두 값의 형이 같으면 단순히 비교만 함.  
2. 두 값의 형이 다르면 두 값을 **동일한 형** 으로 변환하고 비교함.  

  ##### [example]  
1. 99 == "바닐라"   
   99 == NaN  
   => **false**

2. 1 == true  
   1 == 1  
   => **true**

3. undefined == null (두 값 모두 '없는 값'을 비교)  
   => **true**
